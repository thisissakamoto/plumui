# Plum UI

**Plum UI** is a modern, highly customizable React component library built on top of [Ark UI](https://ark-ui.com/) and [Panda CSS](https://panda-css.com/). It provides a set of accessible, reusable, and themeable UI components for building scalable web and mobile applications.

## Features

- **Accessible by default** – built with Ark UI’s accessibility-first approach.
- **Theming and Styling** – Panda CSS recipes for consistent, responsive, and customizable design.
- **Compound Components** – support for advanced UI patterns with context-based composition.
- **TypeScript Support** – fully typed for safe and predictable development.

## Installation

Install Plum UI in your project via npm:

```bash
npm install @plumui/react
# or
yarn add @plumui/react
# or
pnpm add @plumui/react
```
