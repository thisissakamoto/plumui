## How You Can Help

There are several ways you can contribute to making Plum UI better:

- **Enhance documentation**: Fix typos, add missing information, or create new demos
- **Fix issues**: Help resolve bugs or implement missing features
- **Code contributions**: Work on existing issues or propose new features via GitHub Discussions
- **Code quality**: Improve existing code, optimize performance, or enhance reliability

> We value all contributions, regardless of size or scope. Every bit helps!

## Project Architecture

Plum UI v0.x builds upon two key projects in our ecosystem:

- **Ark UI** - Component foundation
- **Zag.js** - Logic and accessibility engine

Our philosophy is to minimize code duplication by leveraging these specialized libraries whenever possible.
